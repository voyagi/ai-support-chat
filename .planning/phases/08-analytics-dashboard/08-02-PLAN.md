**ChatVolumeChart.tsx:**

Recharts AreaChart showing conversations and messages over time.

- Props: `period?: "day" | "week"` (default "day")
- Fetch from `/api/analytics/chat-volume?period=${period}` in useEffect
- Transform dates using `format(new Date(row.date), "MMM d")` from date-fns
- Render `<ResponsiveContainer>` wrapping `<AreaChart>` with two `<Area>` series:
  - Conversations: blue fill/stroke
  - Messages: green fill/stroke
- Include `<XAxis>`, `<YAxis>`, `<CartesianGrid>`, `<Tooltip>`, `<Legend>`
- Handle loading state (show skeleton or spinner while data loads)
- Handle empty state (show "No data yet" message)
- Use Tailwind classes for card wrapper styling
- Dark mode: use CSS variables or explicit dark: classes for text colors
- Chart colors should use fixed hex values (not Tailwind theme colors) since Recharts is SVG-based

Example color scheme:
- Conversations area: stroke="#3b82f6" fill="#3b82f6" fillOpacity={0.1}
- Messages area: stroke="#10b981" fill="#10b981" fillOpacity={0.1}

**AccuracyMetrics.tsx:**

Recharts PieChart (donut style) showing KB accuracy.

- Fetch from `/api/analytics/accuracy` in useEffect
- Render `<PieChart>` with single `<Pie>` using `innerRadius` and `outerRadius` for donut effect
- Two segments: "Answered from KB" (green) and "Fallback" (orange/amber)
- Large percentage number centered below the donut
- Handle zero messages gracefully (show "No data" instead of NaN%)
- Card wrapper with title "Response Accuracy"

**RealtimeMetrics.tsx:**

Live metric cards using Supabase Realtime subscriptions.

- Import `createClient` from `@/lib/supabase/client`
- On mount:
  1. Fetch initial counts:
     - Total conversations: `supabase.from("conversations").select("id", { count: "exact", head: true })`
     - Messages today: same but filtered by `created_at >= today`
     - Total messages (all time): simple count
  2. Subscribe to `conversations` INSERT events (increment total conversations)
  3. Subscribe to `messages` INSERT events (increment messages today if created today, always increment total messages)
- Display three stat cards in a row:
  - Total Conversations (all time)
  - Messages Today
  - Total Messages (all time)
- Use `lucide-react` icons: MessageSquare, MessagesSquare, BarChart3
- Animate count changes with a subtle pulse or number transition
- Clean up subscriptions in useEffect cleanup

**IMPORTANT for Supabase Realtime:** The conversations and messages tables must have Realtime enabled in Supabase Dashboard (Database > Replication > supabase_realtime publication). Document this as a prerequisite but don't fail if Realtime isn't enabled -- the initial fetch will still show correct counts, just without live updates.

**RecentQuestions.tsx:**

Simple list of the 10 most recent user messages.

- Fetch from Supabase directly (client-side): `supabase.from("messages").select("content, created_at").eq("role", "user").order("created_at", { ascending: false }).limit(10)`
- Display as a card with a list of questions, each showing the truncated text and relative timestamp
- Use `formatDistanceToNow` from date-fns for relative timestamps ("2 hours ago", "3 days ago")
- Handle empty state: "No questions yet"
- This serves as a simple ANLY-02 implementation (most common questions) without complex clustering

All components should follow existing project patterns:
- Tailwind for styling (dark: variants for dark mode)
- Consistent card styling: rounded-xl border shadow-sm bg-white dark:bg-gray-800 border-gray-200 dark:border-gray-700
- Use the existing cn() utility from @/lib/cn if combining conditional classes
