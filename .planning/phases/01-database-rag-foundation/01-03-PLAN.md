2. Create `src/lib/embeddings/embeddings.ts`:

Export two functions:
- `generateEmbedding(text: string): Promise<number[]>` — embed a single text string. Uses OpenAI `text-embedding-3-small` model. Returns the embedding vector (1536 floats).
- `generateEmbeddings(texts: string[]): Promise<number[][]>` — embed multiple texts in a single API call (batch). OpenAI supports up to 2048 inputs per request. If `texts.length > 2048`, split into batches and concatenate results.

Import the openai client from `@/lib/openai`. Use `openai.embeddings.create({ model: "text-embedding-3-small", input: texts, encoding_format: "float" })`.

Handle errors: if the API call fails, throw with a descriptive message including the number of texts attempted and the error.

Add a check: if any text exceeds 8191 tokens, log a warning and truncate (embedding model's max input). Use `isWithinTokenLimit` from token-counter.

3. Create `src/lib/rag/similarity-search.ts`:

Export function:
- `searchSimilarChunks(query: string, options?: { threshold?: number; count?: number }): Promise<SimilarChunk[]>`

Where `SimilarChunk` is:
```typescript
export interface SimilarChunk {
  id: string;
  documentId: string;
  documentTitle: string;
  sectionHeading: string;
  content: string;
  chunkPosition: number;
  totalChunks: number;
  similarity: number;
}
```

Implementation:
a. Generate embedding for the query using `generateEmbedding(query)`
b. Call Supabase RPC: `supabase.rpc('match_document_chunks', { query_embedding: embedding, match_threshold: threshold ?? 0.7, match_count: count ?? 5 })`
c. Use the service role client from `createServiceRoleClient()` in `@/lib/supabase/server.ts` (the RPC needs server-side access)
d. Map the snake_case response to camelCase SimilarChunk interface
e. Handle errors: if RPC fails, throw with descriptive message

4. Fix `src/lib/supabase/server.ts` — the `createServiceRoleClient` function currently uses `require()` which is fragile. Refactor to use proper ESM import:

```typescript
    process.env.NEXT_PUBLIC_SUPABASE_URL ?? "",
    process.env.SUPABASE_SERVICE_ROLE_KEY ?? ""
  );

```

Keep the existing `createServerSupabaseClient` function unchanged.
