**Step 3: Create `src/lib/chat/context-builder.ts`**

Two exported functions:

1. `formatRagContext(chunks: SimilarChunk[]): string`
   - Import `SimilarChunk` from `@/lib/rag/similarity-search`
   - Format each chunk as: `[{documentTitle} - {sectionHeading}]\n{content}`
   - Join with `\n\n---\n\n` separator
   - Return formatted string

2. `selectHistoryMessages(messages: Array<{role: string; content: string}>, budgetTokens: number): Array<{role: string; content: string}>`
   - Import `countTokens` from `@/lib/embeddings/token-counter`
   - Per locked decision: take last 10 messages (5 user + 5 assistant pairs)
   - Calculate token count of selected messages
   - If over `budgetTokens`, drop oldest messages from the selection until within budget (keep minimum 2 messages)
   - Log warning if truncation occurs
   - Return the selected messages

Export a constant `MAX_INPUT_TOKENS = 120_000` (reserve 8K from 128K context window for output + safety) and a helper `calculateAvailableBudget(systemPromptTokens: number, ragContextTokens: number): number` that returns `MAX_INPUT_TOKENS - systemPromptTokens - ragContextTokens`.

Use the existing `countTokens` from `@/lib/embeddings/token-counter` (already uses gpt-tokenizer o200k_base encoding).
