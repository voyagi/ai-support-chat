**Part D: prefers-reduced-motion support**

Update `src/app/globals.css` — add at the end:
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

This globally disables animations for users who prefer reduced motion, covering both CSS animations (typing indicator bounce, shake) and transitions (hover states, theme changes). Motion (Framer Motion) respects `prefers-reduced-motion` automatically via its `useReducedMotion` hook internally.
