  <action>
1. Install dependencies:
   ```bash
   npm install next-themes motion
   ```

2. Update `src/app/globals.css` — add AFTER the `@import "tailwindcss";` line:
   ```css
   @custom-variant dark (&:where([data-theme="dark"], [data-theme="dark"] *));
   ```
   Also add a scoped theme transition rule (NOT on `*` — too broad). Instead add a utility class:
   ```css
   .theme-transition {
     transition-property: background-color, border-color, color;
     transition-duration: 200ms;
     transition-timing-function: ease-in-out;
   }
   ```
   Keep the existing `@keyframes shake` animation.

3. Update `src/app/layout.tsx`:
   - Import `ThemeProvider` from `next-themes`
   - Add `suppressHydrationWarning` to `<html>` tag
   - Wrap `{children}` inside `<ThemeProvider>` with these props:
     - `attribute="data-theme"`
     - `defaultTheme="system"`
     - `enableSystem`
     - `storageKey="theme"`
   - Keep existing Inter font and metadata

4. Create `src/components/ui/ThemeToggle.tsx`:
   - Client component (`"use client"`)
   - Import `useTheme` from `next-themes`, `Sun`, `Moon`, `Monitor` from `lucide-react`
   - Use `useState` for mounted state and dropdown open state
   - `useEffect(() => setMounted(true), [])` to avoid hydration mismatch
   - Return `null` until mounted (no SSR render)
   - Render a button showing current theme icon (Sun for light, Moon for dark, Monitor for system)
   - On click, toggle a dropdown with 3 options: Light (Sun icon), Dark (Moon icon), System (Monitor icon)
   - Each option calls `setTheme('light' | 'dark' | 'system')`
   - Active option has `bg-gray-100 dark:bg-gray-700` highlight
   - Dropdown positioned absolute, has `bg-white dark:bg-gray-800 border border-gray-200 dark:border-gray-700 rounded-lg shadow-lg`
   - Close dropdown when clicking outside (useEffect with document click listener)
   - Button styling: `p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800 transition-colors`
   - Export as named export `ThemeToggle`
  </action>

    - `grep -r "@custom-variant dark" src/app/globals.css` finds the variant
    - `grep -r "ThemeProvider" src/app/layout.tsx` confirms wrapper
